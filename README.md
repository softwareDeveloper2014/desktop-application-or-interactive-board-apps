## 1 обзор проекта

видеообзор:  
https://rutube.ru/video/a8efeeca90e0aae2f7dd2995a7d57448/

- обучающие курсы можно приобрести в онлайн магазине компании:  
  https://web.archive.org/web/20201023163058/http://www.nd.ru/catalog/education/



функционал:

- воспроизведение аудио, видео, слайд шоу, анимаций содержащихся в образовательных интерактивных курсах
- управление интерактивными симуляциями различных физических процессов
- возможность использования курсов на интерактивной школьной доске
- возможность использования курсов одновременно несколькими пользователями на одном компьютере


задачи:

- создание готового к продаже продукта. включая написание скриптов для установщиков Windows (installShield Wizard)
- добавление поддержки интерактивной школьной доски в ЦМС курсов
- добавление поддержки нескольких мышей в ЦМС курсов
- разработка интерактивных обучающих курсов по указаниям, макетам и формулам (для симуляций процессов в виртуальных лабораториях) методиста, по школьным учебникам.


инициативность:

- существующие методы разработки, формат контента были значительно усовершенствованы
- были разработаны и успешно внедрены форматы новых более сложных видов интерактивного контента
- существенный объем контента был преобразован в более простой и беспроблемный формат (ручная обработка контента была автоматизирована)
- спроектирована и реализована, на более совершенной версии языка программирования, ЦМС  для интерактивных тестов (было ActionScript 2(без ООП), стало ActionScript 3)
